Carbon credits Test

Steps for testing.
1. Open Postman  
2. Set the GET = https://api.tmsandbox.co.nz/v1/Categories/6327/Details.json?catalogue=false
3. In Tests tab, Copy and paste the contents of "Test Assert" file
4. Click Send and verify the results.
